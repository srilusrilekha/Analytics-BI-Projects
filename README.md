# Analytics-BI-Projects

A collection of enterprise-grade Power BI, business intelligence, analytics, and dashboarding solutions demonstrating data modeling, DAX development, Power Query transformations, KPI reporting, and interactive business analytics.

## Projects

### 📊 Network Performance Analytics Dashboard

Enterprise Power BI reporting solution for monitoring network performance, service management operations, infrastructure health, and operational KPIs across multiple geographic locations.

**Highlights**
- Hybrid Data Model (Star + Snowflake)
- Power Query Transformations
- DAX Measures
- Operational Dashboards
- SLA Monitoring
- Network Performance Analytics

---

### 🛡️ Cyber Security Power BI Analytics

Enterprise cybersecurity reporting solution delivering executive and operational dashboards for Governance, Risk & Compliance (GRC) and Security Operations & Response (SOR).

Built on top of an AWS Data Engineering platform to provide centralized semantic modeling, KPI reporting, and interactive cybersecurity analytics.

**Highlights**
- Star Schema Semantic Model
- Executive Dashboards
- Governance & Compliance Reporting
- Security Operations Analytics
- Power Query & DAX
- Interactive KPI Reporting
