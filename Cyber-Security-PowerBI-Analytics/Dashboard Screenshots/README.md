# Dashboard Screenshots

## Overview

This folder contains representative dashboard samples from the Enterprise Cyber Security Power BI Analytics solution.

The original Power BI application consists of multiple domain-specific dashboards designed to provide operational, governance, and executive visibility into enterprise cybersecurity metrics. To protect confidential business information, only selected dashboards are included in this repository.

These examples demonstrate the reporting design, KPI visualization approach, navigation experience, and interactive capabilities implemented within the Power BI solution.

---

## Dashboard Portfolio

The original Power BI application contains multiple reporting domains supporting different cybersecurity functions across the organization.

Representative dashboard samples included in this repository are:

| Dashboard | Description | Status |
|------------|-------------|--------|
| Governance Risk & Compliance (GRC) | Governance, compliance, supplier risk, audit, and policy reporting | ✅ Included |
| Security Operations & Response (SOR) | Security operations, vulnerability monitoring, remediation, and operational KPIs | ✅ Included |

Additional business domains were developed as part of the enterprise reporting solution but are intentionally omitted from this repository to protect confidential business information.

# 1. Governance Risk & Compliance (GRC)

**File**

`01_Governance_Risk_and_Compliance.png`

### Purpose

Provides business stakeholders with a centralized view of governance, compliance, audit, supplier risk, and policy-related cybersecurity metrics.

The dashboard enables leadership teams to monitor governance performance, identify compliance risks, and evaluate operational trends through interactive KPI visualizations.

### Key Reporting Capabilities

- Executive KPI scorecards
- Compliance tracking
- Supplier governance metrics
- Audit issue monitoring
- Policy violation reporting
- Interactive filtering by business dimensions
- Cross-filtering across visuals

# 2. Security Operations & Response (SOR)

**File**

`02_Security_Operations_and_Response.png`

### Purpose

Provides operational visibility into enterprise security operations by monitoring vulnerability management, remediation activities, SLA compliance, and security response metrics.

The dashboard enables security teams to quickly identify high-risk areas, monitor remediation progress, and prioritize operational activities.

### Key Reporting Capabilities

- Critical vulnerability monitoring
- SLA compliance tracking
- Security operations KPIs
- Operational scorecards
- Drill-through analysis
- Interactive filters and slicers
- Executive and operational reporting
