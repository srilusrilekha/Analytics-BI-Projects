# Dashboard Screenshots

## Overview

This folder showcases representative dashboards from the **Enterprise Cyber Security Power BI Analytics** solution.

The original Power BI application consists of multiple business domains designed to provide executive, operational, and governance visibility into enterprise cybersecurity metrics.

To protect confidential business information, only **two representative dashboards** are included in this repository. These samples demonstrate the overall reporting approach, visualization standards, semantic modeling, and interactive capabilities implemented throughout the solution.

---

## Related Project

The dashboards presented in this repository consume curated datasets generated by the companion cloud data engineering platform.

The complete data engineering implementation—including AWS Glue ETL pipelines, Amazon Redshift, CloudFormation, and GitHub Actions—is documented in the companion repository:

> **Enterprise Cyber Security Data Engineering on AWS**

This repository focuses exclusively on the **Business Intelligence and Analytics** layer built on top of that platform.
