# Data Model

This solution uses a hybrid dimensional model designed to support operational reporting across multiple infrastructure domains.

Key subject areas include:

- Incident Management
- Change Management
- Problem Management
- SLA Monitoring
- Network Performance
- Nodes Utilization
- Device Inventory

Shared dimensions such as Date, Location, Node, and Assignment Group enable cross-functional reporting and drill-through analysis across dashboards.

**Note:**
The simplified model is included to illustrate the overall solution architecture, while the original production data model has been sanitized to protect confidential client information. Sensitive identifiers and proprietary details have been masked for portfolio purposes.
